# 나만의 1년 운동 달력 (Check Your Gain)

이 프로젝트는 1년 단위의 시각적인 달력을 통해, 매일의 운동 및 신체 변화를 기록하고 관리하는 웹 애플리케이션입니다.

## 시작하기 (Getting Started)

1. 터미널에서 프로젝트 폴더로 이동합니다:
   ```bash
   cd fitness-calendar
   ```

2. 패키지를 설치합니다 (이미 설치되어 있을 수 있습니다):
   ```bash
   npm install
   ```

3. 개발 서버를 실행합니다:
   ```bash
   npm run dev
   ```

4. 브라우저에서 `http://localhost:3000`을 엽니다.

## 주요 기능
- **1년 뷰어**: 올해의 모든 날짜를 한눈에 볼 수 있습니다.
- **기록 하기**: 날짜를 클릭하여 체중, 골격근량, 체지방률, 운동 메모를 입력하세요.
- **히트맵**: 기록이 있는 날짜는 달력에 색상으로 표시되어 꾸준함을 시각화합니다.
- **데이터 저장**: 모든 데이터는 브라우저(내 컴퓨터)에 안전하게 저장됩니다.

## 기술 스택
- Next.js 14
- TypeScript
- CSS Modules (Premium Dark Theme)
